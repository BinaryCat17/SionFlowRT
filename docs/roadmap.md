# Roadmap проекта SionFlowRT

## Этап 1: Базовая функциональность (Выполнено ✅)
- [x] Инициализация проекта на Rust.
- [x] Описание структуры графа в JSON.
- [x] Топологическая сортировка и валидация DAG.
- [x] Базовые операции (Add, Mul, Sin).
- [x] Механизм страйдов и Broadcasting.
- [x] Автоматический Loop Fusion.
- [x] Генерация C-кода с OpenMP.

## Этап 2: Расширение вычислительных возможностей (В процессе ⏳)
- [ ] **Операции редукции:** Sum, Mean, Max/Min по осям.
- [ ] **Матричные операции:** MatMul (умножение матриц), Transpose (транспонирование через страйды).
- [ ] **Свёртки (Convolutions):** Реализация 2D свёрток для нейросетевых и графических задач.
- [ ] **Константы:** Поддержка узлов-констант, встроенных прямо в код.

## Этап 3: Оптимизации и управление памятью
- [ ] **Memory Reuse:** Анализ времени жизни тензоров для переиспользования буферов (уменьшение потребления RAM).
- [ ] **SIMD:** Использование заголовочных файлов для явной векторизации (AVX2/AVX-512) в сгенерированном коде.
- [ ] **JIT-компиляция:** Вызов компилятора и подгрузка бинарника прямо во время работы Rust-приложения.

## Этап 4: GPU Backend
- [ ] **WGSL/Naga:** Генерация шейдеров для работы на GPU.
- [ ] **Hybrid Execution:** Автоматическое разделение графа: часть на CPU, часть на GPU.
- [ ] **Buffer Mapping:** Эффективная передача данных между CPU и GPU.

## Этап 5: Интерфейс и экосистема
- [ ] **Визуальное программирование:** Создание GUI для рисования графов.
- [ ] **Python Bindings:** Возможность вызывать скомпилированные графы из Python.
- [ ] **Поддержка ONNX:** Импорт моделей из PyTorch/TensorFlow.
